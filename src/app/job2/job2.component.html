Aplios / Analista de Sistemas

Situación: Mejorar la escalabilidad y eficiencia del desarrollo de servicios críticos de ámbito financiero para el cliente Equifax mediante la creación de microservicios y soluciones automatizadas.

Tarea: Diseñar y desarrollar soluciones microservicios robustos, implementar pruebas automatizadas, y contribuir al desarrollo full-stack con prácticas de desarrollo ágil.

Acción:

    • Desarrollé microservicios en Java utilizando Spring Boot, integrando APIs REST escalables y almacenamiento en caché con Redis, siguiendo principios de arquitectura limpia y desarrollo orientado a pruebas (TDD) con JUnit y Mockito.
    • Implementé orquestadores funcionales que mejoraron la modularidad del sistema, optimizando la reutilización de componentes.
    • Colaboré en el desarrollo backend y frontend de una aplicación generadora de plantillas para servicios .NET, acelerando los despliegues y reduciendo tiempos de configuración.
    • Implementé aplicaciones frontend dinámicas con Angular (TypeScript), aplicación móvil usando React Native creando componentes reutilizables y optimizados.
    • Desarrollé funcionalidades completas de consumo de APIs REST utilizando servicios y modelos Angular, asegurando una integración fluida con microservicios backend desarrollados en Spring Boot
    • Automatice procesos con Jenkins y gestioné tareas con Jira, participando activamente en metodologías Scrum para fomentar la entrega continua.
    • Usé GitHub y herramientas colaborativas de Google para integración continua y control de versiones.

Resultado:

    • Incremento del rendimiento y la capacidad de respuesta del sistema mediante soluciones escalables.
    • Aceleración del desarrollo de servicios en .NET gracias a la creación dinámica con Angular en el frontend y Java en el backend reduciendo el tiempo de desarrollo de 2 semanas a 4 días mediante la creación de componentes reutilizables y patrones de diseño eficientes.
    • Mejora de la calidad del código, asegurando altos estándares de mantenibilidad y cobertura de pruebas.
